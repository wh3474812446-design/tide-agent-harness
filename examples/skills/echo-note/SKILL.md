# echo-note 技能

当用户想"记一笔""存个速记""把这段话记下来"时使用本技能。

## 步骤

1. 取用户要记录的文字内容。
2. 用 `read_file` 读取工作区根目录下的 `notes.md`；如果不存在就准备新建。
3. 在文件末尾追加一行，格式：`- [YYYY-MM-DD HH:mm] <内容>`（时间用当前本地时间）。
4. 用 `write_file` 写回 `notes.md`（保留原有内容 + 新增行）。
5. 用一句话告诉用户已记录，并回显这一行。

## 约定

- 只动 `notes.md` 这一个文件，不要碰其它文件。
- 如果用户没给内容，先问要记什么，别凭空编造。
